DeppFlix

This project will be a RESTful API for an application called “DeppFlix”, which interacts with a non-relational database that stores data about different movies.
I'm building it to get familiar with Node.js and overall server-side programming.

CRUD: Adding Users

/users

Method Type: POST

Description:
Allows users to register by returning a JSON object with specific key values.

"Username" = String
"FavouriteMovies" = Array
"id" = String

Query Parameters:
None

Expected Request Data:
JSON object with data about user.
Required: "Username"

Example:

{
"Username": "Christian",
}

Expected Response Data:
JSON object with data about user incl. automatically generated ID.

Example:

{
"Username": "Christian",
"Password": "MimiIstMeinLiebling123,"
"Email": "christian@email.com,"
"FavouriteMovies": [],
"id": "8a757bb9-2eec-46bf-86f0-443ff742cf3a"
}

Response Status Codes:

201 (Created): New user has been added

400 (Bad Request): User already exists

CRUD: Updating Usernames

/users/:id

Method Type: POST

Description:
Allows users to update their username and returns a JSON object with few key values.

"Username" = String
"FavouriteMovies" = Array
"id" = Integer

Query Parameters:
"_id"

Expected Request Data:
JSON object with data about user

Required: "_id"

Example:

localhost:8080/Christian


{
"Username": "Christian"
}

Expected response data:
JSON object with data about user incl. "_id" with an integer, "Username" with a string and "FavouriteMovies" with an array.

Example:

{
"Name": "Christian",
"Password": "MimiIstMeinLiebling123,"
"Email": "christian@email.com,"
"FavouriteMovies": [],
"DOB":"1985-02-19"
"_id": 8a757bb9-2eec-46bf-86f0-443ff742cf3a,
}

Response status codes:

200 OK: New user has been added

500 Internal Server Error: Something went wrong!

CRUD: Add Movie To Favourites

/users/:id/movieTitle

Method Type: POST

Description:
Allows users to add movie to their favorite list and returns a JSON object with user data incl. an updated FavouriteMovies array.

Query Parameters:
"_id", ":movieTitle"

Expected Request Data:
Required: "_id", ":movieTitle"

Example:

localhost:8080/Christian/movies/Platoon

Expected Response Data:
JSON object with updated list of movies in FavouriteMovies array.

{ Platoon has been added to Christian's FavouriteMovies array }

Response status codes:

200 OK: Updated User's FavouriteMovies

500 Internal Server Error: Something went wrong!

CRUD: Get All Movies

/movies

Method Type: GET

Description:
Allows users to get whole list of movies and returns an array with list of objects.
Each object contains data about movies.

Query Parameters:
None

Expected Request Data:
Required: /movies

Example:

localhost:8080/movies

Expected response data:
An array with list of movies.
Each object contains data about movies and every key comes with a string value.

Example:

[
{
"Title": "Platoon",
"Description": "...",
"Genre": {
"Name": "Drama",
"Description": "..." },
"Director": {
"Name": "...",
"Bio": "...",
"Birth":"July 30, 1970" },
"ImagePath": "...",
"Featured":true
}

Response Status Codes:

200 OK

500 Internal server error: Something went wrong!

Description:

Required: ":movieTitle"

{

"Description": "...",

"Director": { "Name": "...",
"Bio": "...",
"Born": "..."
},
"ImagePath": "...",
"Featured": true }

Response Status Codes:

201 OK: JSON object

500 Internal server error: Something went wrong!
